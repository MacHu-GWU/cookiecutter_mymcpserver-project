Install
------------------------------------------------------------------------------

``cookiecutter_mymcpserver`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install cookiecutter-mymcpserver

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade cookiecutter-mymcpserver
